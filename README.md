# First Friday Club

A React SPA built with Vite, TypeScript, and Tailwind CSS.

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Project Structure

- `src/pages/` - Individual page components
- `src/components/` - Reusable components
- `src/utils/` - Utility functions
- `src/App.tsx` - Main app component that combines all pages in a scrollable layout
